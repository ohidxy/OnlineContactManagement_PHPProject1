Updates: Alpha version release and about next update

Released Version: Alpha 0.1

Date: 19 June 2017

Description:

I am glad to inform you that I am releasing my OCM: Online Contact Management demo alpha version to public. An Alpha version is basically confined into developers but I have chosen to make it public as I believe you deserve testing my system. This is a simple project which will facilitate the developers and others who use fancy contact management but want something more lite and flexible. Also, this will leverage those people who are mostly spending their time in their personal computer. As this is an alpha version, I would like to invite you to test this system with the demo account which can be found in LOGIN PAGE. Please, don't enter any real information until I release the beta version as I am working on some security issues yet. While playing with the system, you may face issues. I would be more than happy to have your BUG REPORT.


Features of this system:

  • Create Unlimited Contacts of Developers and other professionals.
  • Find a contact by filtering by Skill Fields/Roles and Search by name.
  • Create your own preferred unlimited skill fields. (eg. Kotlin Developer, PHP developer etc.)
  • Do you want a feature in this system? I would love to hear from you. Suggest your feedback and suggestions at Feedback & Suggestions

Upcoming Features:

  • Settings: Account Settings, Font Size Adjustment and more!
  • Event Scheduling: Don't forget a single event anymore!
  • Appointment Scheduling: Never miss a single appointment again!

Your bug report will help me to make this system more efficient and secured. This is a completely free2use system.
